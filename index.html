A list of demo and mocked up SocialAPI providers

This website is merely a directory of services that run elsewhere. Other than the Demo Social Service, these demo providers do not run from this domain. You may uninstall the providers later in the Firefox Add-ons Manager.

These are useful as examples and UI testing. Most of these will not work in several ways. In most cases they are not provided by the website linked to. You may install the Demo Social Service, which I use for basic manual UI testing and to illustrate how things are done. This site runs from the github repository for the Demo Social Service.

To install from this directory page you must modify the social.directories preference in Firefox to include the domain , you can do that in about:config. If you do not know how to do that, you probably shouldn't be playing with these examples.

Some of the demos use the mobile websites, you may need to install a WML extension for Firefox.
